# Wearable Fabric - 3D Printed Mesh Insert

This folder contains the print assets for the wearable enclosure used in Human Operator.

The enclosure uses a printed base with an embedded flexible mesh layer. You can use either:

- A compatible synthetic mesh sheet, or
- Thin polyester/nylon fabric with melting point above 250 C.

## Print Parameters

- Pattern size: 5 cm x 10 cm
- First layer height: 0.20 mm
- Subsequent layer height: 0.24 mm
- Material: PLA
- Supports: not required
- Tested printers: Bambu Lab P2S and H2S

## Mesh Insert Procedure

1. Slice the pattern to your target scale.
2. Start the print and pause after layer 2.
3. Place the mesh/fabric carefully on top of the printed layers.
4. Hold the mesh flat with magnets or clips so it cannot shift.
5. Ensure tension is uniform: no bulges, folds, or slack regions.
6. Resume printing and complete the part.

## Practical Notes

- Do a short test print first to verify adhesion and mesh compatibility.
- If edges lift, increase first-layer adhesion and reduce print cooling early in the job.
- Keep mesh clear of moving components before resuming the print.
